# Product Model - AI Website Factory

## Template Collection

### Template 1: Construction / Handwerker

**Target**: Construction companies, craftsmen, tradespeople in Germany + Poland

**Sections**:
- Hero: Background image, headline, CTA to contact
- About: Company story, years experience, certifications
- Services: List of services with icons (renovation, roofing, plumbing, etc.)
- Portfolio: Before/after project gallery
- Testimonials: Customer references
- FAQ: Common questions (warranty, timeline, etc.)
- Contact: Form + map + contact details
- CTA: "Get your free quote" banner

**Pricing**: €750 Basic / €1,200 Pro

### Template 2: Restaurant / Gastronomie

**Target**: Restaurants, cafes, bars in Germany + Poland

**Sections**:
- Hero: Full-width food photo, name, tagline

## Operator Workflow

### New Client - Start to Finish

1. **Discovery Call** (30 min)
   - Identify template type (construction/restaurant)
   - Collect content (text, photos,_logo)
   - Confirm tier (Basic/Pro)

2. **Setup** (30 min)
   - Clone codebase
   - Create Sanity project
   - Configure env vars

3. **Content Entry** (1-2 hrs)
   - Fill in Sanity studio
   - Upload images
   - Configure sections

4. **Customization** (1 hr)
   - Set brand colors in Tailwind config
   - Adjust default text
   - Test forms

5. **Deploy** (15 min)
   - Push to GitHub
   - Connect Vercel
   - Test live site

6. **Handover** (15 min)
   - Show client Sanity Studio
   - Explain how to edit
   - Send login credentials

**Total: ~4 hours per site**
